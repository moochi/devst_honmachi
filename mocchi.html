チートシート予定地

チートシートの表記法

  • $id コミットID、ブランチ名、タグ名のいずれか
  • $file 任意のファイル名
  • $branch 任意のブランチ名
    • mocchi マージの競合を解消 マージの競合を確認する git diff (競合のある差分すべて) git diff --base $file (ベースファイルに対して) git diff --ours $file (あなたの変更に対して) git diff --theirs $file (他人の変更に対して) 競合しているパッチを捨てる git reset --head git rebase --skip 競合を解消してマージする git add $conficting_file (競合を解消した全ファイルに) git rebase --continue solacreative

    表示

    git status
    作業したディレクトリで変更したファイルを表示する
    git diff
    追跡対象のファイルに対する変更を表示する
    git diff $id1 $id2
    $1と$2の差分を表示する
    git log
    変更履歴を表示する
    git log -p $file $dir/ec/tory/
    ファイルの変更履歴を差分付きで表示する
    git blame $file
    ファイルのどこをいつ誰が変更したか表示する
    git show $id
    $idのコミット内容を表示する
    git show $id:$file
    特定$idのファイル$fileを表示する
    git branch
    ローカルブランチを全て表示する
    		 tomix55

    ブランチ

    git branch $branch1 [$branch2]

    $branch1を[$branch2を元に]作成する

    git checkout $branch

    ブランチを切り替える

    取り消し

    最後にコミットした状態に戻す
    git reset --hard
    最後のコミットを取り消す
    git revert HEAD
    指定したコミットを取り消す
    git revert $id
    最新のコミットを修正する
    git commit -a --amend
    ファイル$fileのバージョン$idをチェックアウトする
    git checkout $id $file

    公開

    ローカルの変更をすべてコミットする

    git commit -a

    他の開発者向けにパッチを用意する

    git format-patch origin

    変更をoriginに反映する

    git push

    バージョンやマイルストーンのタグを付ける

    git tag v1.0

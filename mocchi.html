git log

git blame $file

git show $id

git show $id:$file

取り消し

  • 最後にコミットした状態に戻す
    • git reset --hard
  • 最後のコミットを取り消す
    • git revert HEAD
  • 指定したコミットを取り消す
    • git revert $id
  • 最新のコミットを修正する
    • git commit -a --amend
  • ファイル$fileのバージョン$idをチェックアウトする
    • git checkout $id $file
